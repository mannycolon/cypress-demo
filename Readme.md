## Setup Instructions

- yarn add cypress.
- npx cypress open.

## Notes

- We can chane the viewport width and height in a cypress test in the cypress.json file.
- cy.viewport(500, 700)
- cy.viewport('iphone-5')
- cy.viewport('ipad-5')
- cy.viewport('ipad-5', 'landscape')
- cy.viewport('samsung-s10')
